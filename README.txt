Importante extraer correctamente el archivo tar.gz, ya que abrir lo archivos .rkt de forma temporal puede causar problema de funcionamieto.

Para utilizar correctamente archivos .rkt que contiene los problemas a resolver de la tarea, basta con abrirlo y ejecutarlos en DrRacket, y pasar por consola las instrucciones pertinentes para cada funcion.

DISCLAIMER
Debido a que por alguna razon desconocida a algunos computadores de prueba no les permite el ejecutar por la consola de DrRacket sin antes utilizar el siguiente comando:

(require (file "../Tarea4LP_202373102-1/Px.rkt"))

Siendo x el indice del problema. Se recomienda en caso de presentar algun error al intentar usar un caso de prueba utilizar este comando y luego el caso de prueba.
